# PlantBlog

PlantBlog is a small full-stack blogging application built with React, Python Tornado, PostgreSQL, and Docker.

The application allows users to register, log in, create blog posts, update posts, and view published entries on the homepage. The project uses a separate frontend and backend, with Docker used to run the full stack together.

## Main Features

- User registration and login
- Blog post creation
- Blog post editing
- Homepage with published blog entries
- Custom PlantBlog branding and visual theme

## Stack

- Frontend: React
- Backend: Python Tornado
- Database: PostgreSQL
- Environment: Docker Compose

## How It Works

The React frontend handles the user interface.
The Tornado backend handles routes, authentication, and blog logic.
PostgreSQL stores user and blog data.

In short:

`React -> Tornado -> PostgreSQL`
